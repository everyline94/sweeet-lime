sweeet-lime

O design system da lima. Um registry do shadcn: o consumidor recebe o código, não um binário.

Instalar

npx shadcn@latest add https://sweeet-lime.vercel.app/r/botao.json

Os tokens, o Button e as duas fontes (Manrope e JetBrains Mono) chegam junto. Depois é só importar o tokens.css na primeira linha do CSS da aplicação.

Registry por namespace

// components.json do projeto que consome
"registries": {
  "@sweeet-lime": "https://sweeet-lime.vercel.app/r/{name}.json"
}

npx shadcn@latest add @sweeet-lime/blocos
